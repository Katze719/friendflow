# friendflow Support

**Last updated:** July 5, 2026

Need help with friendflow? This page explains where to find support for the app and the official public instance.

## Official public instance

The official public friendflow instance is:

[https://friendflow.site](https://friendflow.site)

If you use the App Store version, this is the default instance unless you connect the app to a custom/self-hosted server.

## Getting help

For general support, bug reports, and feature requests, use the project repository:

[https://github.com/Katze719/friendflow](https://github.com/Katze719/friendflow)

Before opening a new issue, please include:

- What you were trying to do
- What happened instead
- Your device and browser or app version
- Whether you use the official instance or a custom/self-hosted instance

## Account and data questions

If you use the official public instance, account deletion is available in the app under **Account settings -> Delete account**. You can also read the deletion details here:

[/account-deletion](/account-deletion)

For privacy information, see:

[/privacy](/privacy)

## Custom or self-hosted instances

friendflow can be self-hosted. If you use a server operated by someone else, that instance operator is responsible for:

- Account approvals and account deletion
- Password recovery configuration
- Backups and server logs
- Privacy requests for that instance
- Availability and hosting issues

If you are not sure who operates your instance, ask the person who gave you the invite link or server URL.
